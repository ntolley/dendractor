# A novel framework for expanding RNNs with biophysical detail to solve cognitive tasks

_**Author note:** this repository is still a WIP for easy dependency installation and reproduction of paper figures. Contact the author Nicholas Tolley directly for help running code examples._

This repository contains code and examples to reproduce results the preprint [Tolley et al., 2026 bioRxiv](https://www.biorxiv.org/content/10.64898/2026.03.13.711746v1)
